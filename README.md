# 💰 我的财务记录本

一个简洁的个人财务管理工具，支持资产快照、收支记录、投资追踪。

## 功能特点

- 📷 **资产快照**：记录每月资产分布，支持自定义账户
- 📊 **收支追踪**：自动计算月度支出，三色柱状图展示
- 📈 **资产走势**：5条曲线可选（总资产、净资产、负债、流动资金、投资资产）
- 🥧 **资产分布**：饼图展示流动资金、基金、ETF、黄金占比
- 📤 **数据导出/导入**：支持跨设备迁移
- 📱 **响应式设计**：手机电脑均可使用

## 使用方法

1. 访问 https://yanyiding.github.io/my-finance/
2. 首次使用：点击右上角 ⚙️ → 📥 导入数据（可选）
3. 每月月初发工资后：新建快照 + 记录收入
4. 定期导出数据备份

## 技术栈

- 纯前端（HTML + CSS + JavaScript）
- Chart.js 图表库（国内 CDN）
- localStorage 本地存储

## 数据安全

所有数据存储在浏览器本地（localStorage），不会上传到任何服务器。建议定期导出备份。
